About

This site attempts to pay homage to the venerable Roland TR-808 by creating a usable interface to access the original sounds over the internet.

Technical

TR808.online uses Node.JS on the backend and jQuery and jQuery UI on the frontend. It is hosted by DigitalOcean.

Thanks

Mad props to the following projects: Howler.JS, fluent-ffmpeg, ffmpeg, Node.JS, jQuery. Without awesome people creating amazing open source libraries and frameworks like these, sites like this would not be possible.
This would also not be possible without the contributors at Stack Overflow.
